# cell
A cell atlas.

### run locally
`npm start`

### deploy to server
`cell/bin/deployWww`

### install on server
`cell/bin/installWww`

### source-map-explorer
To run source-map-explorer insert this in package.json:
`"source-map-explorer": "^1.5.0",`
This was removed because it causes an npm audit warning on its use of insecure 'open'.
